# Recipe – Flags

All flags in RateGame are **circles**, used for the World Cup and across the app. They
all share the same look, so consistency is the whole game. The library lives in the
[Flags file](/handbook/files).

## The standard

- Every flag is a **circle**.
- Make the size a little **smaller** than you'd expect. That's the RateGame sizing.
- They all carry the same **wavy flag-frame effect**, which is what makes the set feel
  like one family.

## When the component won't cooperate

<Callout type="tip" title="The fallback that always works">
  If the flag components aren't behaving, don't fight them. **Find the flag on Google**,
  then **layer it on top of the wavy flag-frame effect**, the same frame all the others
  use. The result stays perfectly consistent with the rest of the set.
</Callout>
