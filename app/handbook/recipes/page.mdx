# Recipes

Step-by-step guides for the fiddly, repeatable bits of designing RateGame. Each
recipe captures the non-obvious details so the result stays consistent every time.

- **[Achievements](/handbook/recipes/achievements)** — from a Gemini-generated image
  to a proper in-app variant (1:1, light file, dark variant).
- **[Flags](/handbook/recipes/flags)** — keeping every flag consistent: circles with
  the wavy frame, and the fallback when the component won't cooperate.
